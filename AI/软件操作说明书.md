# Aboard 教学白板软件 操作说明书

---

## 一、软件概述

### 1.1 软件简介

Aboard是一款专为教学场景设计的简约Web白板软件。该软件采用HTML5 Canvas技术开发，具有功能简单、部署便捷、使用直观的特点，主要面向国内初高中一体机教学使用设计。

### 1.2 软件特点

- **简洁易用**：界面简约直观，符合教师使用习惯
- **零安装部署**：基于Web技术，打开浏览器即可使用
- **触控优化**：专门针对触控大屏进行优化
- **多语言支持**：支持8种语言界面
- **离线可用**：支持本地部署，无需网络环境

## 二、安装与部署

### 2.1 在线使用

直接访问以下网址即可使用：
- 主版本：https://aboard.pp.ua
- 测试版：https://dev-aboard.619.pp.ua

### 2.2 本地部署

**方法一：Python服务器**
```bash
python3 -m http.server 8080
```

**方法二：Node.js服务器**
```bash
npx http-server -p 8080
```

**方法三：PHP服务器**
```bash
php -S localhost:8080
```

部署后访问 `http://localhost:8080` 即可使用。

### 2.3 云平台部署

支持一键部署到以下平台：
- Vercel
- GitHub Pages
- Cloudflare Pages

## 三、界面介绍

### 3.1 主界面布局

软件主界面分为以下几个区域：

1. **画布区域**：中央区域，用于绑图和书写
2. **工具栏**：底部区域，包含主要绘图工具
3. **控制按钮区**：右上角，包含导出、缩放、翻页等功能
4. **属性面板**：点击工具后弹出，用于设置工具属性

### 3.2 工具栏功能

| 图标 | 功能 | 说明 |
|------|------|------|
| ↩️ | 撤销 | 撤销上一步操作（最多50步） |
| ↪️ | 重做 | 恢复撤销的操作 |
| ✏️ | 笔 | 自由绑图工具 |
| ✋ | 移动 | 移动画布位置 |
| 🧹 | 橡皮 | 擦除画布内容 |
| 🗑️ | 清空 | 清空整个画布 |
| 🎨 | 背景 | 设置背景颜色和图案 |
| ➕ | 更多 | 展开更多功能菜单 |
| ⚙️ | 设置 | 打开设置面板 |

### 3.3 控制按钮区

| 按钮 | 功能 |
|------|------|
| ⏬ | 导出画布为图片 |
| ➕/➖ | 缩放画布大小 |
| 🔢 | 显示/输入缩放比例 |
| ⛶ | 全屏模式 |
| ◀️/▶️ | 翻页（分页模式下） |

## 四、基本操作指南

### 4.1 绘图操作

**自由绑图**
1. 点击工具栏中的"笔"工具
2. 在画布上按下并拖动即可绑图
3. 松开完成一笔

**选择颜色**
1. 点击"笔"工具打开属性面板
2. 选择预设颜色或使用取色器自定义颜色

**调整粗细**
1. 在属性面板中拖动粗细滑块
2. 支持1-50px范围调节

**选择笔触类型**
- 普通笔：标准笔触
- 铅笔：灰色质感带颗粒纹理
- 圆珠笔：压力感应模拟
- 钢笔：变宽笔触效果
- 毛笔：毛边效果和笔锋感觉

### 4.2 擦除操作

**使用橡皮**
1. 点击工具栏中的"橡皮"工具
2. 在画布上按下并拖动进行擦除

**调整橡皮属性**
- 形状：圆形或方形
- 大小：10-50px可调

**清空画布**
1. 点击"清空"按钮
2. 确认后清空所有内容

### 4.3 移动和缩放

**移动画布**
- 方法一：点击"移动"工具，拖动画布
- 方法二：按住Shift键拖动画布

**缩放画布**
- 鼠标：Ctrl + 滚轮
- 触控：双指捏合
- 按钮：点击 ➕/➖ 按钮
- 双击：自动缩放/还原

### 4.4 撤销与重做

- **撤销**：Ctrl + Z 或点击↩️按钮
- **重做**：Ctrl + Y 或点击↪️按钮
- 支持最多50步历史记录

## 五、高级功能

### 5.1 背景设置

**设置背景颜色**
1. 点击"背景"工具
2. 选择预设颜色或使用取色器

**设置背景图案**
- 空白：无图案
- 点阵：规则点阵图案
- 方格：方格线图案
- 田字格：中文书写格
- 英语四线格：英文书写格
- 五线谱：音乐记谱用
- 坐标系：数学坐标系

**上传背景图片**
1. 点击"上传图片"选项
2. 选择本地图片文件
3. 使用蓝色控件调整位置、大小和旋转

### 5.2 形状绘制

**绘制形状**
1. 选择形状工具
2. 选择形状类型（直线、矩形、圆形）
3. 在画布上拖动绘制

**线条样式设置**
- 实线、虚线、点线
- 波浪线、双线、三线

### 5.3 教具功能

**直尺工具**
1. 在"更多"菜单中选择直尺
2. 拖动直尺调整位置
3. 沿直尺边缘可画直线

**三角板工具**
- 60°三角板：30-60-90度直角三角形
- 45°三角板：45-45-90度等腰直角三角形
- 支持旋转、缩放和多实例

### 5.4 计时器功能

**启动计时器**
1. 在"更多"菜单中选择"计时"
2. 设置计时模式（正计时/倒计时）
3. 设置时间长度
4. 点击开始

**计时器设置**
- 自定义字体颜色和背景颜色
- 选择提示音或上传自定义音频
- 设置循环播放次数
- 支持全屏显示

### 5.5 点名器功能

**添加名单**
1. 手动输入姓名
2. 或导入Excel/CSV文件

**随机点名**
1. 点击"点名器"
2. 点击开始随机选择
3. 显示选中结果

### 5.6 计分板功能

**添加队伍**
1. 输入队伍名称
2. 点击添加

**计分操作**
- 点击 ➕ 增加分数
- 点击 ➖ 减少分数
- 支持自定义步进值

### 5.7 图片插入

**插入图片**
1. 在"更多"菜单中选择"插入图片"
2. 选择本地图片文件
3. 图片显示在画布上

**编辑图片**
- 拖动：移动位置
- 控制点：调整大小
- 旋转控件：旋转角度
- 翻转按钮：水平/垂直翻转

## 六、分页功能

### 6.1 启用分页模式

1. 打开设置面板
2. 找到"分页模式"选项
3. 开启分页功能

### 6.2 选择画布尺寸

预设尺寸：
- A4（横向/竖向）
- A3（横向/竖向）
- B5（横向/竖向）
- 16:9 宽屏
- 4:3 标准
- 自定义尺寸

### 6.3 页面管理

- **添加页面**：点击 ➕ 按钮
- **切换页面**：点击 ◀️/▶️ 按钮
- **删除页面**：在页面管理中删除

---

## 七、导出功能

### 7.1 导出图片

1. 点击 ⏬ 导出按钮
2. 选择导出格式（PNG/JPEG）
3. 选择导出范围：
   - 当前页
   - 全部页面
   - 指定页面
4. 输入文件名
5. 点击确认导出

### 7.2 文件命名

- 默认包含时间戳
- 多页导出自动添加页码后缀

## 十、常见问题

### Q1: 如何保存画布内容？
A: 软件支持自动保存功能，使用IndexedDB技术存储。刷新页面时会提示是否恢复上次内容。

### Q2: 支持哪些图片格式？
A: 支持常见图片格式，包括PNG、JPEG、GIF、WebP等。

### Q3: 如何清除所有数据？
A: 在设置中可以找到"清除数据"选项，或在浏览器设置中清除网站数据。

### Q4: 触控屏无法正常使用？
A: 请确保使用支持的浏览器（Chrome/Edge），并检查触控驱动是否正常。

### Q5: 导出图片模糊？
A: 尝试调整画布缩放比例，或选择更高的导出质量。
