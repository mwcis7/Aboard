## 一、软件内容简介

### 1.1 软件基本信息

- **软件名称**：Aboard 教学白板软件
- **软件版本**：V1.0
- **开发完成日期**：2024年
- **软件类型**：Web应用程序（网页应用）
- **运行环境**：现代Web浏览器（Chrome、Firefox、Safari、Edge等）
- **开发语言**：HTML5、CSS3、JavaScript（ES6+）
- **软件规模**：约34,576行代码

### 1.2 软件功能概述

Aboard是一款专为教学场景设计的简约Web白板软件，主要面向国内初高中一体机教学使用。该软件具有功能简单、部署便捷、使用直观的特点，能够满足教师课堂批注和教学演示的需求。

#### 核心功能模块：

**（1）多样化绘图工具**
- 支持多种笔触类型：普通笔、铅笔、圆珠笔、钢笔、毛笔
- 形状绘制工具：直线、矩形、圆形等几何形状
- 丰富的线条样式：实线、虚线、点线、波浪线、双线、三线
- 智能橡皮擦：支持圆形和方形，大小可调（10-50px）
- 8种预设颜色及自定义取色器
- 笔触粗细1-50px可调

**（2）专业教学背景**
- 多种教学背景图案：空白、点阵、方格、田字格、英语四线格、五线谱、坐标系
- 可移动坐标系原点功能
- 自定义背景图片上传，支持拖动、旋转、缩放和翻转
- 8种预设背景颜色及自定义颜色

**（3）赛博教具功能**
- 直尺工具：两种样式，支持沿边缘画直线
- 三角板工具：60°三角板（30-60-90度）和45°三角板（45-45-90度）
- 智能尺寸自动缩放、自由拖动移动、旋转角度调整
- 多实例支持，双指缩放和旋转

**（4）分页画布管理**
- 多页面管理，支持A4、A3、B5、16:9、4:3等预设尺寸
- 自定义画布尺寸
- 智能分页切换和导出

**（5）计时器与时间显示**
- 正计时和倒计时模式
- 自定义字体和背景颜色
- 多种提示音选择
- 时区自动检测
- 全屏显示模式

**（6）辅助教学工具**
- 点名器：支持Excel/CSV导入名单
- 计分板：课堂互动计分
- 图片插入：支持拖动、缩放、旋转和翻转

**（7）国际化支持**
- 支持8种语言：简体中文、繁体中文、英语、日语、韩语、法语、德语、西班牙语
- 自动检测浏览器语言

**（8）数据管理**
- 导出PNG/JPEG图片
- IndexedDB本地自动保存
- 配置导入导出

## 二、软件运行情况说明

### 2.1 运行环境要求

#### 硬件环境
- **处理器**：支持HTML5的现代处理器
- **内存**：建议2GB及以上
- **显示设备**：支持触控屏幕（推荐）或普通显示器
- **输入设备**：鼠标、触控笔或触摸屏

#### 软件环境
- **操作系统**：Windows 7及以上、macOS 10.12及以上、Linux、iOS、Android
- **浏览器要求**：
  | 浏览器 | 最低版本 |
  |--------|----------|
  | Chrome | 最新版本 |
  | Firefox | 最新版本 |
  | Safari | 最新版本 |
  | Edge | 最新版本 |
  | iOS Safari | 支持 |
  | Chrome Mobile | 支持 |

### 2.2 软件部署方式

本软件支持多种部署方式：

**（1）在线部署**
- Vercel平台一键部署
- GitHub Pages部署
- Cloudflare Pages部署

**（2）本地部署**
```bash
# 使用Python启动本地服务器
python3 -m http.server 8080

# 使用Node.js启动
npx http-server -p 8080

# 使用PHP启动
php -S localhost:8080
```
访问地址：`http://localhost:8080`

### 2.3 软件操作说明

#### 基本操作
| 操作 | 方法 |
|------|------|
| 绘图 | 点击/触摸画布并拖动 |
| 选择工具 | 点击底部工具栏相应按钮 |
| 改变颜色/粗细 | 点击工具按钮，在弹出的属性面板中调节 |
| 撤销/重做 | Ctrl+Z / Ctrl+Y 或点击按钮 |
| 缩放画布 | Ctrl+滚轮 或点击缩放按钮（50%-无上限） |
| 移动画布 | 点击"移动"工具或按住Shift键拖动 |
| 全屏 | F11 或点击全屏按钮 |
| 清空画布 | 点击"清空"按钮（有确认提示） |

#### 高级功能
- 所有控制面板均可拖动调整位置
- 支持图片作为背景
- 选择工具可复制或删除画布元素
- 分页模式支持多页切换和管理

### 2.4 技术架构

#### 技术栈
- **HTML5 Canvas** - 高性能2D图形渲染
- **Vanilla JavaScript** - 无框架依赖，模块化架构
- **CSS3** - 现代化UI设计

#### 核心模块
- DrawingEngine - 核心绘图引擎
- ShapeDrawingManager - 形状绘制管理
- HistoryManager - 历史记录管理（撤销/重做）
- BackgroundManager - 背景管理
- SelectionManager - 选择管理
- SettingsManager - 设置管理
- TimerManager - 计时器管理
- TimeDisplayManager - 时间显示管理
- ExportManager - 导出管理
- TeachingToolsManager - 教具管理
- RandomPickerManager - 点名器管理
- ScoreboardManager - 计分板管理
- StorageManager - 存储管理（IndexedDB）
- I18n - 国际化模块

#### 性能优化特性
- Canvas desynchronized模式减少渲染延迟
- 按需加载（Lazy Loading）提升首屏速度
- requestAnimationFrame优化动画性能
- GPU加速渲染优化
- 触控事件优化，支持多点触控
